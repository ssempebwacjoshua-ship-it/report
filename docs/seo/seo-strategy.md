# SSAMENJ Public SEO Strategy

## Goal

Make the public SSAMENJ marketing site easy for Google to crawl, understand, and index for the main product and keyword landing pages:

- `/`
- `/products`
- `/report-lab`
- `/smart-pages`
- `/nfc`
- `/pricing`
- `/demos`
- `/contact`
- `/rentflow`
- `/cashless-canteen`
- `/stayos`

## Strategy

1. Keep each public page focused on one primary keyword theme.
2. Give each page a unique title, meta description, and canonical URL.
3. Expose only real public URLs in the sitemap.
4. Use honest structured data that matches visible page content.
5. Keep the public site linkable from the homepage, footer, product pages, and primary navigation.
6. Avoid indexing private app routes, API routes, and internal dashboard pages.

## Public Page Themes

- Home: SSAMENJ Technologies, school software, and the full product family.
- Report Lab: school report card generation.
- Smart Pages: digital school document workflows.
- NFC: gate access, attendance, and cashless school workflows.
- Cashless Canteen: Kids Wallet and NFC-linked school canteen payments.
- StayOS: property operations keyword landing page for the RentFlow workflow.
- RentFlow: rentals and property management.

## Search Console Priorities

- Submit `https://ssamenj.vercel.app/sitemap.xml`.
- Inspect the main product URLs after deployment.
- Verify that Google sees the new landing pages as real pages, not homepage fallbacks.
- Monitor crawl status for sitemap fetches and coverage.
