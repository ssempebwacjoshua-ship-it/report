# UI Page Template

Use this structure for new pages and major page rewrites.

## Page Structure

- Page header.
- Primary action.
- Search/filter bar.
- Main data area.
- Empty, loading, and error states.
- Permission denied state.
- Mobile layout.
- Dangerous action confirmations.

## Dashboard Checklist

Every dashboard should answer:

- What changed today?
- What needs action?
- What is pending?
- What is risky?
- What can the operator do now?

## Measurement Rule

Follow `03-ui-source-of-truth.md`. Document exact colors, spacing, sizing, typography, radii, shadows, breakpoints, button/input heights, table row heights, and card measurements when introducing reusable UI.
