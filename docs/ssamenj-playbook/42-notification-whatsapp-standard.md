# Notification And WhatsApp Standard

Messaging must be intentional, permissioned, and safe.

## Rules

- Use approved SSAMENJ numbers.
- Use prefilled message formats for public links.
- Avoid sensitive data in public WhatsApp links.
- Require operator approval for sensitive messages.
- Prevent duplicate bulk messages.
- Audit demand, payment, and outreach messages.

## Sensitive Data Rule

Do not put report tokens, payment secrets, private guest details, student private data, or internal IDs in public WhatsApp URLs.
