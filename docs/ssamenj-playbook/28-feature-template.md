## Report Lab Notes

For Report Lab features, explicitly define school/tenant scope and whether the feature touches marks import, scan/OCR upload, report generation, student data, passport photos, public verification, Smart Pages, or print/report layout. Preserve exact measurement rules for colors, spacing, padding, gaps, widths, heights, typography, shadows, radii, and breakpoints.
