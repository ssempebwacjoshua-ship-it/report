# School Connect Smart Pages Official Documentation Pack

School Connect Smart Pages is SSAMENJ Technologies' school document-intelligence module for notices, circulars, timetables, programmes, meeting minutes, exam schedules, school policies, and other general school documents.

Important: Smart Pages is separate from Report Lab. It is not the Report Lab marksheet import flow.
