# School Connect Smart Pages User Manual

## 1. Log In

1. Sign in with your school account.
1. Open the school app.
1. Use the Smart Pages area from the main workspace.

Smart Pages uses school-authenticated access for school documents.

## 2. Open Smart Pages

1. Go to the Smart Pages page.
1. Review the document list.
1. Open an existing document or create a new one.

The main page shows:

- Your documents
- Credit balance
- Billing link
- Create document button

## 3. Create a New Document

1. Click New Document.
1. Enter a title.
1. Create the document.

You can start with a blank workspace and add content later, or upload a file for extraction.

## 4. Upload a Document

1. Open the document workspace.
1. Click upload.
1. Select a file.
1. Wait while Smart Pages reads the file.
1. Review the extracted content when it appears.

Supported file types in the current codebase:

- PDF
- Images
- CSV
- XLS
- XLSX

Word documents are not supported yet.

## 5. Review the Extracted Information

After upload, Smart Pages may show:

- Extracted text
- Suggested document type
- Detected tables
- Unclear items
- A review warning if the scan needs attention

Use this review step before generating the final document.

## 6. Use Templates

1. Open the template picker in the workspace.
1. Choose the template that matches your document.
1. Pick the summary style if you are using the summary template.
1. Generate the polished output.

Current school template examples include:

- School notice
- School circular
- Timetable
- School programme
- Meeting minutes
- Exam schedule
- Letter to parents
- Attendance sheet
- Extract to table
- Publish secure link
- Generate in bulk

## 7. Edit or Correct Content

1. Click into the extracted text area.
1. Fix names, dates, and wording.
1. Save your review edits.
1. Generate the final page when the content looks right.

You can also type directly into the workspace when no file has been uploaded yet.

## 8. Save and Organize Work

1. Save your draft as you go.
1. Use versions if you need to compare changes.
1. Use collections when you want to group records for bulk generation.

## 9. Publish or Print

1. Open the finished document.
1. Use Print to create a paper copy.
1. Use Download PDF if you want a file copy.
1. Use Publish Secure Link if you want a shareable public link.

You can optionally protect the public link with a password.

## 10. Common Mistakes

- Uploading the wrong file type.
- Expecting the system to guess unclear text.
- Skipping the review step.
- Confusing Smart Pages with Report Lab marksheet import.
- Forgetting to save before generating a new version.

## 11. What To Do When Extraction Fails

1. Check whether the file is readable and complete.
1. Try a clearer scan or photo.
1. Retry extraction.
1. If the page is handwritten, try high accuracy extraction.
1. If the file is Word format, convert it to PDF or image first.

## 12. Difference Between Smart Pages and Report Lab

Use Smart Pages for school documents such as notices, circulars, schedules, minutes, and policies.

Use Report Lab for the marksheet and report workflows.

They are separate modules.

## 13. Human Review Before Use

Always review:

- Names
- Dates
- Times
- Class names
- Totals
- Any text that will be shared publicly
