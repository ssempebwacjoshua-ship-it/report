# School Connect NFC / Kids Wallet Official Documentation Pack

School Connect NFC, also referred to as Kids Wallet in school-facing workflows, is the school operations module for NFC wristbands, attendance, gate security, wallet balances, canteen charging, fee holds, tag issuing, and offline register sync.

Important: this module is separate from Report Lab and Smart Pages. It is not a document-generation or marksheet product.

- School owners and headteachers: start with the product overview and onboarding guide.
