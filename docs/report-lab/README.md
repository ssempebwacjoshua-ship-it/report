# School Connect Report Lab Documentation Pack

School Connect Report Lab is the school reporting module in the SSAMENJ Technologies suite. It helps schools manage students, classes, streams, subjects, marks import, report generation, parent report links, and report verification.

| Document | Audience | Purpose |

| [03-school-onboarding-guide.md](./03-school-onboarding-guide.md) | Onboarding team, school admins | What a school must prepare before setup and first use |
| [04-technical-documentation.md](./04-technical-documentation.md) | Developers, support engineers | Architecture, data model, auth, deployment, and environment notes |
