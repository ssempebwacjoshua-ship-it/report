# Style Rules

- Style changes must follow SSAMENJ design tokens and measured values.
- Do not add random colors, shadows, spacing, or font sizes.
- If a new value is needed, document why and consider adding it to `src/styles/ssamenjDesignTokens.ts`.
- Prefer exact reusable variables and tokens over one-off CSS.
- Do not silently normalize inconsistent values; document inconsistencies and make small safe changes only.
