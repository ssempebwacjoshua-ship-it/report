# Component Rules

- UI must follow `docs/ssamenj-playbook/03-ui-source-of-truth.md`.
- Do not invent new card, button, table, spacing, shadow, color, or typography styles if existing reusable patterns exist.
- Capture exact measurements where relevant: gap, padding, width, height, radius, shadow, font size, line height, and breakpoints.
- Components must include empty, loading, and error states where applicable.
- Preserve mobile and PWA behavior.
